About this template

The following gives a brief introduction and tutorial on how one can use this template to create easy and quick but still eye-appealing slides. Currently, LaTeX-Beamer is still the go-to document preparation system when it comes to writing slides, especially in the scientific community. Hence, this template aims to achieve two things:

  • Making slides presentation more interactive
  • Making the experience of writing slides less painful and fast

Vertical Slides

One of the most useful features of reveal.js is the fact that one can add vertical slides. This would be usedful when it comes to regrouping ideas. This can be achieved by just adding a section tag inside the section as follows: 


                <section>
                  <section>Vertical Slide 1</section>
                  <section>Vertical Slide 2</section>
                </section>

Another Slide

One can even highlight specific lines. 


                <section>
                  <section>Vertical Slide 1</section>
                  <section>Vertical Slide 2</section>
                </section>

Animation

Perhaps one of the main advantages of reveal-js over Beamer is the fact that one can add videos.
